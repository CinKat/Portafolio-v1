General concepts of selectors in CSS

One of the first rules to style our website is the use of selectors. In CSS selectors are used to select the HTML elements that you want to style.

Selectors tend to fall into five categories:

  1. Simple or basic selectors
  2. Attribute selectors
  3. Pseudo-class selectors
  4. Pseudo-element selectors
  5. Combiner Selectors

Let's start learning the basics of selectors.

Basic CSS selectors

These types of selectors are enough to be able to create a stylish web page.

  • CSS element selector (type): Select HTML elements based on element names or semantic names. 
    •                         /* selecting all p elements */
                        
                            p {
                                text-align: center;
                                color: red;
                            }
                        
                        /* selecting all article elements */
                        
                            article {
                                font-size: 14px;
                                line-height: 1.1px;
                            }
  • CSS id selector: If a single element needs a unique style, we can give it an id. This can only have a single value and can only be used once per page. To be able to select we prepend the name of the id with a (#). 
    •                         
                            #contenedor-flex {
                                display: flex;
                                gap: 1rem;
                            }
  • Selector of CSS classes: It is one of the most common ways to select a style, to use it a period is included. + class name. 
    •                         
                            .mb-32 {
                                margin-bottom: 32px;
                            }
  • Universal selector: This selector gives style to the entire entire page, it is used with the *. 
    •                         
                            * {
                                box-sizing: border-box;
                            }

Attribute selectors

These types of selectors are enough to be able to create a stylish web page.

Attribute selectors allow you to select elements based on whether a certain attribute is present or not.

It is to be able to add styles to the attributes of an HTML element such as hrf, src, etc. The way to apply it individually is by calling the name of the element, the name of the attribute and its value if it has to be able to identify it. 

                    
                        a[title] { }

Pseudo-class and pseudo-element selectors

First, pseudo-class selectors select items based on a certain state. You may have seen things like: hover or: active. These are the states of the elements on your page. You can select items based on whether that item is in the specified state.

While pseudoelement selectors select parts of an element, for this two double points are used: a difference of pseudoclasses. 

                    /* Pseudo-class */
                    
                        a: hover {}
                    
                    /* Pseudo-element */
                    
                        p::first-line { }

Combiner selectors

These selectors can combine more than one CSS selector. We have the:

  1. Descendant selectors
  2. Child selectors
  3. Adjacent sibling selectors
  4. General sibling selectors

In order to understand its use, let's remember that HTML follows a hierarchy of a family tree. This means that there is a parent element that can contain children, and children can have children, and so on, like a family tree. 

                    
                        article > p { }

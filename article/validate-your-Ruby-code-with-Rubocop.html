Validate your Ruby code with Rubocop

Ruby recommends the best practices so that we can write a clean code so that other programmers can understand it, through these recommendations we have The Ruby style guide.

The guide is divided into several related guideline sections that are based on the professional experience of the editors, comments and suggestions from members of the Ruby community, and various programming resources.

Use Rubocop to validate our code

Ruby's style guide is extensive, both in the rules defined on structure, syntax, classes, exception handling and many more sections.

Learning all the rules is somewhat complicated, so there are code analyzers like Rubocop that verify your code structure and compliance with each of the rules defined in the style guide.

Rubocop is extremely easy to install and like any other gem we can install it in two ways: global or local.
